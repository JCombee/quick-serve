Quick Serve
===========

## Usage

```

Usage:   quick-serve [directory]
Version: 0.1.0

Description:

  Your quick command line server!

Options:

  -h, --help                 - Show this help.
  -V, --version              - Show the version number for this program.
  -p, --port     <port>      - The port number for the local server.      (Default: 8080)
  --host         <hostname>  - The host name for the local server.        (Default: "localhost")

```
